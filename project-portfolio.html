Portfolio Website

A responsive personal portfolio to showcase my projects, skills, education, and contact details.

Overview

This project is a personal portfolio website built using HTML, CSS, and JavaScript. It highlights my key projects, technical skills, education background, and links to my resume and LinkedIn profile.

Key Features

  • Clean, responsive layout that works on both desktop and mobile.
  • Smooth scrolling navigation for sections like Projects, Skills, Education, and Contact.
  • Dedicated Resume button to open my CV directly.
  • Project cards with separate detail pages for deeper explanations.

Tech Stack

  • HTML5, CSS3
  • Vanilla JavaScript for interactivity and smooth scrolling
