AI Room Guard Project

A multi-modal AI security agent that combines voice activation, face verification and LLM-based interaction.

Overview

This project implements an AI “room guard” that can listen for a wake word, recognize who is in front of the camera, and then respond using a conversational LLM. The system works as an intelligent security assistant that can distinguish trusted users from unknown people and react accordingly.

Key Features

  • Developed a multi-modal guard agent in Python by integrating voice activation using SpeechRecognition, high-accuracy face recognition with DeepFace, and a conversational LLM via the Groq API.
  • Engineered a robust face verification module by fine-tuning the VGG-Face model and using a custom cosine similarity threshold to clearly separate trusted users from unknown faces.
  • Implemented an autonomous monitoring loop that listens for a voice trigger, captures webcam frames, and decides whether to issue a verbal warning or a friendly greeting.

How It Works

  • The system stays idle until a wake word is detected through the microphone.
  • On activation, the webcam captures a face image and passes it through the VGG-Face model.
  • The embedding is compared against stored trusted embeddings using cosine similarity.
  • If the user is unknown, the system triggers a security warning; if trusted, it greets the user and can answer questions via the LLM.

Tech Stack

  • Python
  • DeepFace, VGG-Face
  • SpeechRecognition for voice input
  • Groq API (LLM)
  • OpenCV for webcam capture
