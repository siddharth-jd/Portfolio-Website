Hand Written Digit Classifier

Convolutional Neural Network in PyTorch to classify MNIST digits with high accuracy.

Overview

This project builds a convolutional neural network (CNN) in PyTorch to classify handwritten digits from the MNIST dataset. The focus is on designing an efficient architecture, using augmentation to improve generalization, and analyzing performance with standard metrics.

Key Features

  • Designed a multi-layer CNN trained for around 10 epochs using GPU acceleration.
  • Applied data augmentation (rotation, translation, normalization) to improve robustness.
  • Achieved strong classification accuracy on the MNIST test set.
  • Generated classification reports and confusion matrices for detailed error analysis.

Training Pipeline

  • Loaded MNIST images and labels using PyTorch DataLoaders.
  • Applied transforms for normalization and augmentation.
  • Trained the CNN using cross-entropy loss and an optimizer like Adam/SGD.
  • Evaluated performance on a held-out test set and visualized predictions.

Tech Stack

  • Python
  • PyTorch
  • Matplotlib / Seaborn for visualization
