Contacts List – React Single Page App

A responsive contacts manager built with React, Tailwind CSS and Framer Motion.

Overview

This project is a single-page contact management application built from scratch with React. It focuses on creating a smooth, modern user experience with animations, theming and real-time search while keeping all data in the browser using local storage.

Key Features

  • Built a fully responsive contacts UI using React and Tailwind CSS, with smooth transitions and micro-interactions powered by Framer Motion.
  • Implemented all core contact features: add, edit, delete contacts, mark favorites, and group contacts alphabetically.
  • Added live search, favorites filtering and dark/light mode using React state.
  • Persisted all user data, including contacts, profile photos and settings, in localStorage so the app works across page reloads without a backend.
  • Deployed the app using Vercel and maintained the codebase on GitHub.

How It Works

  • Contacts are stored in a React state array and mirrored to local storage on every change.
  • Forms handle creating and editing contacts, including optional avatar/image uploads.
  • Search input and filters derive a computed list which is rendered and grouped by alphabet.
  • A theme toggle switches between dark and light Tailwind classes globally.

Tech Stack

  • React
  • Tailwind CSS
  • Framer Motion
  • localStorage (no backend)
  • Vercel for deployment, GitHub for version control
