Heart Disease Modeling Project

Predicting heart disease risk using classical ML models, PCA, clustering, and regression analysis.

Overview

This project explores a full data science pipeline for heart disease prediction and analysis. I used classical machine learning models to classify patients, dimensionality reduction to understand feature importance, and clustering to discover hidden patient segments.

Key Features

  • Trained Logistic Regression and Random Forest models to classify heart disease risk.
  • Applied PCA to reduce dimensionality while retaining 95% of variance.
  • Performed K-Means clustering on PCA-reduced data to group similar patient profiles.
  • Used Multiple Linear Regression to predict serum cholesterol and interpret key factors.

Workflow

  • Cleaned and preprocessed clinical features (missing values, scaling, encoding).
  • Ran PCA to compress the feature space for visualization and clustering.
  • Trained classifiers and evaluated them using accuracy and other metrics.
  • Visualized clusters and model outputs to understand risk patterns.

Tech Stack

  • Python
  • scikit-learn
  • Pandas, NumPy
  • Matplotlib / Seaborn for EDA and plots
